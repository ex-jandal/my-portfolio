### **ملخص سريع (TL;DR)**
لغة **Rust** هي لغة أنظمة متعددة الأنماط صُممت من أجل **الأداء والسلامة**. توفر اللغة تحكماً منخفض المستوى يضاهي لغة C++، لكنها تستخدم نظام "ملكية" (Ownership) صارم لضمان سلامة الذاكرة والتعامل الآمن مع الخيوط (thread-safety) أثناء وقت التحميل، دون الحاجة إلى مجمع نفايات (Garbage Collector).

---

## **لماذا لغة Rust؟**
تعتبر Rust اللغة المفضلة للمطورين الذين يحتاجون إلى كود **فائق السرعة** و**مضاد للرصاص** في آن واحد. تُستخدم على نطاق واسع في محركات المتصفحات (مثل Firefox)، وأنظمة التشغيل، والبرمجيات الخلفية (Backends) عالية الأداء.

### **الميزات الرئيسية**
* **تجريدات صفرية التكلفة (Zero-Cost Abstractions):** يمكنك كتابة كود عالي المستوى ومعبر، يتم تحويله عند التجميع إلى كود آلة بكفاءة تضاهي لغة التجميع (Assembly) المكتوبة يدوياً.
* **فاحص الاستعارة (The Borrow Checker):** جزء من المترجم يضمن عدم وجود "مؤشرات معلقة" (dangling pointers) أو "تنازع على البيانات" (data races).
* **بدون مجمع نفايات (No Garbage Collector):** على عكس Java أو Go، تدير Rust الذاكرة أثناء وقت التحميل، مما يعني عدم وجود "توقفات" غير متوقعة أثناء التنفيذ.
* **أداة Cargo:** أداة بناء ومدير حزم رائد في الصناعة، يجعل إدارة الاعتماديات والاختبارات أمراً في غاية السهولة.

---

> تخيل الذاكرة مثل السيارة. في C++، تملك المفاتيح ويمكنك القيادة في أي مكان، لكنك قد تنسى ركنها أو تعطي المفاتيح لشخصين في وقت واحد (مما يسبب حادثاً). في **Rust**، المترجم هو **موظف ركن السيارات (Valet)**؛ فهو يتتبع بدقة من يملك المفاتيح. يمكنك "إعارة" السيارة (borrowing)، لكن الموظف لن يسمح للسيارة بمغادرة الموقع حتى تعود المفاتيح بأمان.

---

### **مقتطف برمجي (Snippet)**
تمنعك Rust من استخدام البيانات بعد "نقلها" (moved)، مما يمسك بفئة كاملة من الأخطاء البرمجية قبل تشغيل التطبيق.

```rust
fn main() {
    let name = String::from("Abu_Jandal");
    
    greet(name); 
    
    // println!("أهلاً مجدداً، {}", name); 
    // ^ خطأ! تم "نقل" المتغير 'name' إلى دالة greet.
    // المترجم يمنعك من استخدامه هنا مجدداً.
}

fn greet(s: String) {
    println!("مرحباً بك في النظام، {s}");
}
```
