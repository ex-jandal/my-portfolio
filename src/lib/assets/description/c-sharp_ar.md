### **ملخص سريع (TL;DR)**
لغة **C#** (سي شارب) هي لغة حديثة كائنية التوجه طورتها شركة Microsoft. وتعتبر اللغة الأساسية لنظام **.NET** البيئي، وتُستخدم على نطاق واسع في بناء كل شيء، بدءاً من تطبيقات الويب للمؤسسات وصولاً إلى الألعاب عالية الأداء وعابرة المنصات باستخدام محرك **Unity**.

---

## **العملاق متعدد الاستخدامات**
توازن C# بين قوة لغة C++ وسهولة استخدام لغة Java. هي لغة **ثابتة النوع (statically-typed)** تعمل على بيئة التشغيل المشتركة (CLR)، وتوفر مجموعة غنية من الميزات مثل LINQ، وasync/await، ودعم قوي لإصدارات البرامج.

### **الميزات الرئيسية**
* **الذاكرة المُدارة:** مثل جافا، تستخدم C# **مجمع نفايات (Garbage Collector)** عالي الأداء للتعامل مع الذاكرة، ولكنها تسمح أيضاً بكتابة كود "غير آمن" (unsafe) للتحكم المباشر في الذاكرة عند الحاجة.
* **LINQ (الاستعلام المدمج في اللغة):** ميزة فريدة تتيح لك إجراء استعلامات على البيانات من المجموعات، أو قواعد البيانات، أو ملفات XML مباشرة داخل قواعد اللغة.
* **البرمجة غير المتزامنة:** كانت C# رائدة في استخدام الكلمات المفتاحية `async` و `await` مما يجعل كتابة الكود غير المعطل (non-blocking) أمراً في غاية السهولة.
* **التكامل مع Unity:** إذا كنت مهتماً بتطوير الألعاب، فإن C# هي المعيار الصناعي لكتابة السكريبتات في محرك Unity.

> فكر في البرمجة بلغة C# مثل العيش في **منزل ذكي**. يمكنك القيام بكل شيء يدوياً إذا أردت، ولكن في معظم الأوقات، تضاء المصابيح تلقائياً (Garbage Collection)، ويعدل منظم الحرارة نفسه (تحسينات وقت التشغيل). لقد صُممت لتسمح لك بالتركيز على "العيش" (بناء الميزات) بدلاً من "الصيانة" (إدارة الذاكرة).

### **مقتطف برمجي: الخصائص و LINQ**
تُعرف قواعد لغة C# بأنها "نظيفة" وتقلل من الكود الزائد (boilerplate) مقارنة بلغة جافا.

```csharp
using System;
using System.Linq;
using System.Collections.Generic;

public class Program {
    public static void Main() {
        var dev = new Developer { 
            Name = "Abu_Jandal", 
            Stack = new List<string> { "Rust", "Zig", "C#" } 
        };

        var favorite = dev.Stack.First(s => s == "Rust");

        Console.WriteLine($"أهلاً يا {dev.Name}! أداتك المفضلة هي {favorite}.");
    }
}

public class Developer {
    public string Name { get; set; }
    public List<string> Stack { get; set; }
}
```
