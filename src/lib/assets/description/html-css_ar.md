### **ملخص سريع (TL;DR)**
تُعد لغتا **HTML** و **CSS** الثنائي التأسيسي لشبكة الويب. توفر **HTML** (لغة توصيف النص الفائق) **البنية** والمعنى لصفحة الويب، بينما توفر **CSS** (أوراق الأنماط المتتالية) **العرض المرئي** والتنسيق.

---

## **الهيكل العظمي والمظهر خارجي**
في تطوير الويب، تعمل هاتان اللغتان جنباً إلى جنب؛ فلا يمكن بناء موقع ويب حديث بدونهما معاً.

### **HTML: المهندس المعماري**
تحدد HTML *ماذا* يوجد في الصفحة. تستخدم "الوسوم" (Tags) لتسمية المحتوى حتى يعرف المتصفح كيفية التعامل معه.
* **المعنى الدلالي (Semantic Meaning):** تخبر الوسوم مثل `<header>` و `<footer>` و `<article>` محركات البحث وقارئات الشاشة بالضبط عما يمثله كل جزء من الصفحة.
* **السمات (Attributes):** توفر معلومات إضافية، مثل وجهة الرابط (`href`) أو مصدر الصورة (`src`).

### **CSS: خبير التنسيق**
تحدد CSS *كيف* تبدو عناصر HTML. فهي تستهدف عناصر HTML وتطبق عليها القواعد الجمالية.
* **نموذج الصندوق (The Box Model):** يتم التعامل مع كل عنصر كصندوق مستطيل له هوامش (margins)، حدود (borders)، حواف داخلية (padding)، ومحتوى.
* **محركات التخطيط (Layout Engines):** تستخدم CSS الحديثة نظامي **Flexbox** و **Grid** لإنشاء تخطيطات معقدة ومتجاوبة تعمل على الهواتف وشاشات الـ 4K على حد سواء.
* **المتغيرات والدوال:** تدعم CSS الحديثة (CSS3+) المتغيرات، الدوال الرياضية، والرسوم المتحركة مباشرة في المتصفح.

> فكر في صفحة الويب كأنها كائن بشري؛ **HTML** هي **الهيكل العظمي**: تحدد مكان الرأس والذراعين والساقين. أما **CSS** فهي **الجلد والملابس والشعر**: تحدد لون العينين، وتنسيق القميص، والأسلوب العام. (وإذا كنت تتساءل، فإن **JavaScript** ستكون هي **العضلات** التي تجعل الشخص يتحرك!).

### **مقتطف برمجي (Snippet)**
إليك كيفية ربط البنية بالتنسيق:

```html
<div class="profile-card">
  <h1>Abu_Jandal</h1>
  <p>مطوّر فول ستاك (Full-stack Developer)</p>
</div>

<style>
  .profile-card {
    background-color: #282828; /* Gruvbox Dark */
    color: #ebdbb2;           /* Gruvbox Light */
    padding: 20px;
    border-radius: 8px;
    border: 2px solid #d79921;
    text-align: center;
  }
  
  h1 { 
    margin: 0; 
    font-family: 'Fira Code', monospace; 
  }
</style>
```
